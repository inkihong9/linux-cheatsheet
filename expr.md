# expr
Performs basic mathematical calculations. Mathematical operators that can be used are adding `+`, subtracting `-`, multiplying `*` and dividing `/`
## examples
```
$ expr 32 - 8
24
$ expr 3500 * 12
42000
$ expr 32-8
32-8
$ expr 123+456
123+456
$ 
```
## notes
* only works with integers, and all results are rounded down to the nearest integer
* all terms and operators must be separated by spaces (see example above)
