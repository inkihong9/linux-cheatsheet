# `mv`
moves a file or directory to a new location
# usage
`$ mv source-file target-path`
# notes
* can be used to rename files by passing in the new name as the 2nd argument
* can move AND rename a file in a single command
# examples
to move a file to a new location (current directory to absolute path):

`$ mv moveme.txt /home/linux-user/`
